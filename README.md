# Improvement Board

A team improvement tracking tool based on Management 3.0's Improvement Dialogues and Copilot Programs — capture problems, run structured dialogues, assign peer coaches, and track progress.

Part of the [Agile Tools](https://github.com/bthos) suite built on Management 3.0 and ICAgile source materials.

## Stack
React 18 · TypeScript · Vite · Tailwind CSS · react-i18next (EN/RU)

## Development
```bash
npm install
npm run dev
```

## Deploy
GitHub Pages via GitHub Actions on push to `main`.
